Project Title : SyncUp
Description: All in one financial advisory app
Team Name : 4T1S (Kinsley Panashe Ngirazi, Asser Tawfik, Yoon Har Ley, Khaira Radiyah, Mishal Ismail)
Technologies used : Claude, Node, Gitbash, Visual StudioCode
Challenge and Approach : Debugging the AI API feature; Prompt Claude over and over again until it was fixed.
Usage instructions: If running with visual studio code, must type "npm start"

AI-Assisted Features:

- Resume improvement (improveResume) - Uses Claude API to rewrite experience text professionally.
- Meeting notes summary (generateMeetingSummary) - Uses Claude API to extract summary and action items from transcripts.
- Referral insights (renderReferralAiInsight) - Uses Claude API to analyze referral network and give strategic advice.
- Expense analysis (runAiExpenseAnalysis) - Uses Claude API to analyze spending patterns, identify top category, and give cost-saving recommendations.
- Learning coach recommendations (renderLearning) - Uses AI logic to identify skill gaps and suggest learning paths.
- Follow-up suggestions (renderFollowupSuggestions) - Uses AI logic/heuristics to detect cold matches and prompt action.
- Daily brief (renderDailyBrief) - Uses AI logic to summarize day and highlight overdue contacts.
- Partner match scores - Mock data likely AI-generated.
- Seed data - Partners, events, learning logs likely AI-generated.
- CPD badge unlocking - Rule-based AI logic to track progress and auto-unlock badges.
